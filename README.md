# CNN Image Classification - Multi-Class Weather Dataset

## Model Performance
- Train Accuracy: 93.52%
- Test Accuracy: 92.90%

## Architecture
- Conv2D layers: 32, 64, 128 filters
- MaxPooling2D layers
- Dense: 256 units with Dropout (0.5)

## Data Augmentation
- Rotation ±20°, Shift ±20%, Zoom ±20%, Horizontal Flip

## Dataset Split
- Training: 70%, Test: 15%, Validation: 15%
